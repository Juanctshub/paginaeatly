Revolución Tecnológica Libre

¡Apostamos por el futuro colaborativo! Utilizamos Open Food Facts para darte acceso a millones de productos, OpenStreetMap + Leaflet para guiar tus pasos, y la increíble Overpass API para detectar vida en restaurantes locales. Sin costes de API inflados, solo la pureza de datos reales y precisos al servicio de tu salud.

Geolocalización con Alma

Nos importa tu paz mental. Usamos Nominatim para encontrarte el restaurante perfecto mediante geocoding inverso, pero sin rastrear jamás quién eres. Tu ubicación es un dato sagrado que solo sirve para protegerte, de forma efímera y radicalmente segura.

El Poder de lo Nativo

Exprimimos cada gota de potencia de tu smartphone. Con Web Speech API hablas con tu comida y el motor neural responde al instante, mientras que las Web Notifications te alertan de riesgos antes de que ocurran. Latencia cero, protección infinita.

Persistencia Híbrida

Tu seguridad no depende de una señal WiFi. Nuestra sincronización blinda tus datos en SQLite local y los respalda bajo encriptación Prisma.

Privacidad de Grado Médico

Compromiso 01

Encriptación Extremo a Extremo

Tus restricciones alimentarias se consideran datos de salud sensibles. Solo tú y las personas a las que autorices explícitamente pueden ver este perfil.

Compromiso 02

Sin Venta de Datos

Eatly no vende ni comparte tus perfiles de dieta con anunciantes o terceros bajo ninguna circunstancia comercial.

Preguntas Frecuentes

¿Es Eatly apto para alérgicos crónicos?

Absolutamente. Nuestra IA está entrenada con bases de datos toxicológicas y la API de Open Food Facts para detectar no solo ingredientes principales, sino trazas por contaminación cruzada.

¿Qué pasa si no tengo internet en el súper?

Eatly almacena una caché local de tus 5,000 productos más comunes. El escaneo básico por código de barras funciona offline.
